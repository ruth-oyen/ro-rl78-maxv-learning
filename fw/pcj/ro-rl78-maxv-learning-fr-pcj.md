# Code Generator Settings for PCJ

If you do not want to modify the RL78 firmware, you do not need this document. Just download
`RO_RL78_MAXV_LEARNING_FW.mot` to the RL78 using Renesas Flash Programmer.

The source code in this repository is generated automatically with the following settings, and some files are modified by hand.
If you want to modify this firmware:

- Install e2studio on your PC
- Create a project
- Apply the following settings
- Modify the generated code
- Build the project
